# Installation
1. Clone or download the project
```bash
git clone https://github.com/yukic993/FTEC5660_Project
```

``` bash
cd FTEC5660_Project
```

2. Install dependencies
``` bash
pip install -r requirements.txt
```

# Run the Project

```bash
streamlit run streamlitFrontEnd.py
```
